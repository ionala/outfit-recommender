# Outfit Recommender

A web-based outfit recommendation system that helps users manage their wardrobe and receive outfit recommendations based on weather and clothing categories.

## Features

- User authentication (Register, Login)
- Manage personal wardrobe
- Browse default wardrobe
- Outfit recommendation
- User profile management
- Upload profile picture

## Technologies Used

### Frontend
- React
- Vite
- Axios
- CSS

### Backend
- Flask
- Flask-JWT-Extended
- Flask-SQLAlchemy
- PostgreSQL

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
flask run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Backend:

- DATABASE_URL
- SECRET_KEY
- PROFILE_UPLOAD_FOLDER

Frontend:

- VITE_API_URL
